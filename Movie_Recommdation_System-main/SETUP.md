If you want to display actual movie posters, you can optionally add a free TMDB API key:

#### Step 1: Get a Free TMDB API Key (Optional)

1. Go to [TMDB Website](https://www.themoviedb.org/)
2. Create a free account (takes 2 minutes)
3. Navigate to [API Settings](https://www.themoviedb.org/settings/api)
4. Click "Request an API Key"
5. Select "Developer" as the type
6. Fill in the application details
7. Copy your API key

#### Step 2: Set the API Key (Optional)

**Windows (PowerShell):**
```powershell
$env:TMDB_API_KEY="your_api_key_here"
python app.py
```

**Linux/Mac:**
```bash
export TMDB_API_KEY=your_api_key_here
python app.py
```
